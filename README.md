Computer-vision app to identify travel items and advise whether they are allowed in cabin, must be checked in, or forbidden.

Repository structure (work-in-progress):
- data/            -> raw / downloaded image subsets
- notebooks/       -> exploration & training notebooks
- models/          -> saved model weights
- src/             -> code: training, inference, utils
- app/             -> real-time demo (OpenCV / Streamlit / mobile)
- requirements.txt -> pip-installable dependencies

See the project plan in the repo root for step-by-step tasks.
